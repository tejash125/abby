# Abby — Artificial Baby

A long-term research project exploring how a baby might think and learn, replicated in an artificial simulated environment.

The project questions foundational assumptions about computation, cognition, and learning — from programming paradigms to analog computing, neuromorphic chips, and custom hardware. The goal is to research, learn, and explore from the ground up.

## P0 Target

An online learning system that continuously learns, integrates memory, and can understand analog clocks, the concept of time, and draw pictures of telling time accurately.
